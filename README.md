# 🛍️ Myntra Sales Dashboard (Power BI)

An interactive **Power BI dashboard** designed to analyze Myntra’s sales data and generate meaningful business insights.
This project focuses on **sales trends, customer behavior, product performance, and revenue analysis**.

---

## 📌 Project Overview

This dashboard provides a complete view of Myntra’s e-commerce performance:

* 📊 Sales & Revenue Analysis
* 🧑‍🤝‍🧑 Customer Insights
* 🛍️ Product & Category Performance
* 📍 Region-wise Sales Distribution
* 📈 Time-based Trends & Forecasting

Power BI dashboards help businesses make **data-driven decisions using interactive visuals and KPIs**.
